# 2023 F2E 第二關總統即時開票全台地圖
## 作品說明
根據設計稿中的說明，主要分成三塊
1. 搜尋縣市，區域，鄉鎮
2. 顯示投票率, 當年參選黨的投票數比較
3. 台灣地圖互動 - 這部分為了挑戰自己，使用D3.js來產地圖，因此會與原搞不同

## 系統說明
專案直接開啟 index.html 即可使用

## 檔案說明
* COUNTY_MOI_1090820.json - 台灣地圖的TopoJSON
* taiwan-election.xls - 台灣的選舉紀錄

## 使用技術
* Bootstrap
* Chart.js
* D3.js
* Xlsx.js
* Topojson.js
